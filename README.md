# AI Resume

这个仓库是由两个原始主仓库按历史合并得到：

- `frontend/app/` 来自原前端仓库 `master`
- `server/` 来自原后端仓库 `master`

本仓库保留了两个原仓库的提交历史，不是快照拼接。

原始仓库来源：

- 前端：`https://github.com/QssssY/ai_resume.git`
- 后端：`https://gitee.com/yang93311/ai-resume.git`

目录结构：

- `frontend/app/`：Vue 3 前端应用
- `server/`：Spring Boot 后端服务

说明：

- 当前根仓库是在保留两个原始 `master` 历史的前提下，将前端导入到 `frontend/app/`，后端导入到 `server/`
- 后续根级说明文件和忽略规则作为合并仓库自己的提交追加在历史顶部
