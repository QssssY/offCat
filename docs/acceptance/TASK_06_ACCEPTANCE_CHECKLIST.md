## 安全验收

| 检查项 | 通过标准 | 状态 |
|--------|----------|------|
| 敏感配置 | 仓库中不包含明文 API Key、密码、Token、OSS 凭据 | 待验收 |
| 鉴权 | 用户端和管理端受保护接口均需 Token | 待验收 |
| 权限 | 管理端需要管理员角色，用户数据按当前用户隔离 | 待验收 |
| 输入校验 | 表单、上传、社区内容、AI 配置均有校验 | 待验收 |
| 文件安全 | 上传与图片访问防止路径穿越和非法对象访问 | 待验收 |
| 日志脱敏 | 日志不输出敏感字段明文 | 待验收 |
| XSS 防护 | 富文本、社区内容、PDF 导出 HTML 有净化处理 | 待验收 |
