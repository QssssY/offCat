# offcat 演示指南

本文档用于准备一次完整的 offcat 本地演示，覆盖用户端、管理端、AI 配置、简历、面试、社区和通知等当前能力。

## 演示前准备

### 基础服务

| 服务 | 要求 |
|------|------|
| MySQL | 已创建 `ai_resume` 数据库并导入 `server/db/schema.sql` |
| Redis | 可连接，用于缓存、限流和部分业务状态 |
| RabbitMQ | 可连接，用于简历诊断队列兜底 |
| 后端 | `server` 启动在 `http://localhost:8080` |
| 前端 | `frontend/app` 启动在 `http://localhost:3000` |
| Chrome | 已配置后端 PDF 导出所需 Chrome 路径 |

### 启动命令

```bash
cd server
mvn spring-boot:run
```

```bash
cd frontend/app
npm install
npm run dev
```

### 账号准备

- 普通用户：用于演示注册、登录、简历诊断、面试、社区、设置。
- 管理员用户：`role=9`，用于演示 `/admin` 管理后台。
- 会员用户可选：用于演示会员套餐、VIP 权益和配额差异。

账号密码、JWT、数据库密码、AI Key 等敏感值不要写入文档或提交记录。

## 演示路线

### 1. 首页与登录

演示入口：

- `/`
- `/login`
- `/version-logs`

演示要点：

- 公共首页、版本日志可未登录访问。
- 注册成功后自动初始化用户基础数据和配额。
- 登录后用户端路由由 JWT 控制访问。

### 2. 用户仪表盘与设置

演示入口：

- `/dashboard`
- `/settings`
- `/membership`

演示要点：

- 用户状态、会员状态、额度和最近活动。
- 设置中心包含账号资料、面试偏好、自定义 AI、安全、隐私、数据管理、反馈、外观、通知、新手引导、会员与配额。
- 自定义 AI 支持 OpenAI 兼容配置、模型发现、连通性测试、TTS 配置和用量查询。

### 3. 简历诊断

演示入口：

- `/resume/upload`
- `/resume/result/:taskId`
- `/resume/history`

演示要点：

- 上传 PDF/DOC/DOCX 简历，服务端创建诊断任务。
- 前端轮询任务状态和阶段进度。
- 后端根据负载直接异步处理或进入 RabbitMQ。
- 诊断结果包含评分、基础信息提取、问题分析和优化建议。
- 失败任务可以重试，失败会退还相关配额。

### 4. JD 匹配与 AI 润色

演示入口：

- 简历诊断结果页
- 模板编辑器

演示要点：

- 提交 JD 文本生成岗位匹配分析。
- 对简历内容进行 AI 润色。
- 润色结果可进入模板编辑链路。
- 支持 PDF、PNG、DOCX 导出，其中 PDF 导出依赖后端 Headless Chrome。

### 5. 模拟面试

演示入口：

- `/interview/entry`
- `/interview/session/:sessionId`
- `/interview/report/:sessionId`
- `/interview/history`

演示要点：

- 岗位、难度、面试模式、反馈模式、交互方式可配置。
- 支持普通、压力、岗位定向、大厂 HR、技术负责人、外企面试官等模式。
- SSE 流式返回 AI 面试官回复。
- 语音面试可结合浏览器语音识别与云端 TTS。
- 结束面试后生成综合评分、六维评分和报告。
- 面试报告可分享至社区。

### 6. Offer 助手与成长中心

演示入口：

- `/offer`
- `/growth`

演示要点：

- Offer 助手提供薪资谈判模拟和话术脚本。
- 成长中心展示面试维度雷达、薄弱项和成长建议。

### 7. 社区

演示入口：

- `/community`
- `/community/post/:postId`
- `/community/my`

演示要点：

- 帖子支持分类、图文内容、评论、回复、点赞、收藏。
- 支持上传社区图片。
- 支持分享面试报告链接。
- 我的活动展示发帖、评论、点赞、收藏和收到的互动。

### 8. 通知

演示入口：

- `/notifications`
- 顶部通知入口

演示要点：

- 通知列表、未读数量、单条已读、全部已读、删除。
- SSE 实时通知推送。
- 简历、面试、系统公告、活动、更新、维护等通知类型。

### 9. 管理后台

演示入口：

- `/admin/login`
- `/admin/dashboard`

演示要点：

- 管理端使用独立 Token 和角色校验。
- 看板、监控、用户权益、配额、消耗记录。
- 岗位、Prompt、AI 引擎、TTS 配置。
- 会员套餐、订单、通知、版本日志。
- 社区审核、反馈管理、审计日志、成长配置。

## 常见演示问题

| 问题 | 排查方向 |
|------|----------|
| 前端接口 401 | 检查 Token 是否存在、是否过期、是否登录了对应用户端或管理端 |
| 简历任务一直处理中 | 检查后端日志、AI 配置、RabbitMQ、任务恢复调度 |
| 流式面试无输出 | 检查浏览器网络面板、SSE 响应、AI Provider 配置和限流 |
| PDF 导出失败 | 检查 `app.pdf.chrome-path`、Chrome 是否可执行、后端日志 |
| TTS 无音频 | 检查系统 TTS 或用户 TTS 配置、连通性测试结果 |
| 社区图片不可访问 | 检查 OSS 配置、图片登记状态、对象 Key 是否有效 |

## 演示边界

- 当前项目支持 mock/real AI 模式切换，演示时应明确当前使用的模式。
- 外部 AI、TTS、OSS 服务依赖各自网络与账号配置。
- 生产环境敏感配置必须通过环境变量或安全配置注入，不应写入仓库。
