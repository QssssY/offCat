## 安全检查

| 检查项 | 要求 |
|--------|------|
| 硬编码密钥 | 不提交 API Key、JWT 密钥、数据库密码、OSS 凭据 |
| 鉴权 | 受保护接口必须校验 JWT，管理端必须校验管理员角色 |
| 越权 | 用户只能访问自己的任务、会话、通知、设置和社区私有数据 |
| 输入校验 | 注册、登录、上传、评论、帖子、AI 配置、管理端表单必须校验 |
| 文件安全 | 上传类型、大小、路径访问、社区图片对象 Key 需要校验 |
| XSS | 富文本、社区内容、简历导出 HTML 需要净化 |
| 限流 | 注册、密码重置、简历上传、面试、Offer 等关键接口启用限流 |
| 日志 | 日志不得输出明文 Token、密码、API Key、密保答案 |
